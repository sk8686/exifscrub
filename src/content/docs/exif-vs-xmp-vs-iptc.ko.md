사진 메타데이터는 세 가지 주요 형식으로 존재합니다: EXIF, XMP, IPTC. 각각 다른 용도를 가지며 다른 개인정보 영향을 미칩니다.

## 빠른 비교

| 기능 | EXIF | XMP | IPTC |
|------|------|-----|------|
| 생성 주체 | 카메라 제조사 | Adobe | 뉴스 기관 |
| 자동 생성 | 예 | 부분 | 아니오 |
| GPS 포함 | 예 | 가능 | 아니오 |
| 카메라 정보 포함 | 예 | 가능 | 아니오 |
| 캡션 포함 | 아니오 | 예 | 예 |
| 개인정보 위험 | **높음** | 중간 | 중간 |
| 형식 | 바이너리 | XML (텍스트) | 바이너리 |

## EXIF — 카메라의 기록

EXIF는 카메라나 스마트폰에 의해 자동으로 생성됩니다. 가장 일반적이고 개인정보에 민감한 유형의 메타데이터로, GPS 좌표, 카메라 정보, 타임스탬프, 기기 시리얼 번호, 섬네일을 포함합니다.

**개인정보 위험: 높음** — EXIF는 사용자의 모르는 사이 자동 추가되는 GPS 데이터와 기기 식별자를 포함하므로 주요 관심사입니다.

## XMP — Adobe의 현대적 표준

XMP는 Adobe에서 개발되었으며 XML 형식을 사용합니다. 모든 EXIF 및 IPTC 데이터와 편집 기록, 평점, 키워드, 플러그인의 사용자 정의 메타데이터를 포함할 수 있습니다.

**개인정보 위험: 중간** — XMP는 편집 기록을 포함할 수 있으며 종종 GPS 좌표 같은 민감한 EXIF 데이터를 복제합니다.

## IPTC — 뉴스 표준

IPTC 메타데이터는 주로 뉴스 기관, 스톡 사진 에이전시, 전문 사진작가가 사용합니다. 캡션, 키워드, 사진작가 크레딧, 저작권 고지, 연락처 정보를 포함합니다.

**개인정보 위험: 중간** — IPTC는 이름, 연락처 정보, 저작권 세부사항을 포함할 수 있습니다.

Lorsque vous partagez une photo, il est possible que vous partagiez bien plus que l'image elle-même. Les métadonnées EXIF intégrées à vos photos peuvent révéler votre localisation exacte, l'appareil que vous avez utilisé et le moment où la photo a été prise. Voici comment les supprimer.

## La méthode la plus simple : utilisez notre outil en ligne gratuit

Le moyen le plus rapide de supprimer les données EXIF est avec notre [EXIF Scrub gratuit](/fr/) :

1. **Glissez-déposez** vos photos (ou cliquez pour parcourir, ou collez depuis le presse-papiers)
2. **Examinez** les risques de confidentialité trouvés dans chaque photo
3. **Cliquez sur « Nettoyer »** pour supprimer toutes les métadonnées
4. **Téléchargez** les photos nettoyées

Tout se passe dans votre navigateur — vos fichiers ne quittent jamais votre appareil. Pas d'inscription, pas de téléversement, aucun risque.

## Supprimer les données EXIF sur iPhone

iOS intègre une suppression EXIF lors du partage :

1. Ouvrez l'application **Photos**
2. Sélectionnez la ou les photos que vous souhaitez partager
3. Appuyez sur le bouton **Partager**
4. Appuyez sur **Options** en haut
5. Désactivez **Localisation**
6. Partagez la photo

**Limitation** : cela supprime uniquement les données GPS, pas le modèle d'appareil ni les horodatages. Pour une suppression complète, utilisez notre [outil en ligne](/fr/).

## Supprimer les données EXIF sur Android

Android n'a pas de suppression EXIF intégrée. Vos options :

- Utilisez notre [outil basé sur le web](/fr/) — fonctionne sur tout navigateur Android
- Installez une application dédiée comme « Photo EXIF Editor » depuis le Play Store
- Certaines applications de galerie disposent de fonctionnalités d'édition de métadonnées

## Supprimer les données EXIF sur Windows

1. Clic droit sur l'image → **Propriétés**
2. Allez dans l'onglet **Détails**
3. Cliquez sur **Supprimer les propriétés et les informations personnelles**
4. Choisissez « Créer une copie avec toutes les propriétés possibles supprimées »
5. Cliquez sur **OK**

**Limitation** : Windows ne peut pas supprimer toutes les balises EXIF. Pour un nettoyage approfondi, utilisez notre [outil basé sur le navigateur](/fr/).

## Supprimer les données EXIF sur macOS

1. Ouvrez l'image dans **Aperçu**
2. Allez dans **Outils** → **Afficher l'inspecteur** (⌘I)
3. Cliquez sur l'onglet **(i)**, puis sur l'onglet **Exif**
4. Cliquez sur **Supprimer les métadonnées** en bas

**Limitation** : Aperçu ne supprime pas tous les types de métadonnées. Pour une suppression complète, utilisez notre [outil en ligne](/fr/).

## Ce qui est supprimé vs ce qui est conservé

**Supprimé** : coordonnées GPS, marque/modèle d'appareil, horodatages, numéros de série, informations logicielles, miniatures, informations de copyright

**Conservé** : orientation de l'image (évite une rotation incorrecte), profil de couleur (garde des couleurs précises), dimensions de l'image, données de pixels réels

## Quand devez-vous supprimer les données EXIF ?

Toujours supprimer les données EXIF avant :
- La publication de photos sur les réseaux sociaux (certaines plateformes les suppriment, beaucoup non)
- L'envoi de photos à des inconnus par e-mail ou applications de messagerie
- Le téléversement de photos sur des forums ou sites de petites annonces
- Le partage de photos dans un contexte professionnel (journalisme, juridique)
- La soumission de photos pour des concours ou publications

사진을 공유할 때 이미지만 공유하는 것이 아닐 수 있습니다. 사진에 포함된 EXIF 메타데이터는 정확한 위치, 사용한 기기, 사진 촬영 시간을 드러낼 수 있습니다. 제거하는 방법은 다음과 같습니다.

## 가장 쉬운 방법: 무료 온라인 도구 사용

EXIF 데이터를 제거하는 가장 빠른 방법은 저희의 [무료 EXIF Scrub](/ko/)를 이용하는 것입니다:

1. 사진을 **드래그 앤 드롭**합니다 (또는 클릭해 찾아보거나 클립보드에서 붙여넣기)
2. 각 사진에서 발견된 **개인정보 위험 확인**
3. **"Clean" 클릭**하여 모든 메타데이터 제거
4. **다운로드** 정리된 사진

모든 과정이 브라우저 내에서 이루어집니다 — 파일은 절대 기기를 벗어나지 않습니다. 가입 없이, 업로드 없이, 위험 없이.

## 아이폰에서 EXIF 데이터 제거

iOS에는 공유 시 내장된 EXIF 제거 기능이 있습니다:

1. **사진** 앱 열기
2. 공유하고 싶은 사진 선택
3. **공유** 버튼 탭
4. 상단의 **옵션** 탭
5. **위치** 끄기
6. 사진 공유

**제한 사항**: GPS 데이터만 제거되며, 카메라 모델이나 타임스탬프는 남습니다. 완전 제거를 위해선 [온라인 도구](/ko/)를 이용하세요.

## 안드로이드에서 EXIF 데이터 제거

안드로이드에는 내장된 EXIF 제거 기능이 없습니다. 가능한 옵션:

- 저희의 [웹 기반 도구](/ko/) 사용 — 모든 안드로이드 브라우저에서 작동
- Play 스토어에서 "Photo EXIF Editor" 같은 전용 앱 설치
- 일부 갤러리 앱은 메타데이터 편집 기능 제공

## Windows에서 EXIF 데이터 제거

1. 이미지 우클릭 → **속성**
2. **세부사항** 탭으로 이동
3. **속성 및 개인정보 제거** 클릭
4. "모든 가능한 속성이 제거된 복사본 만들기" 선택
5. **확인** 클릭

**제한 사항**: Windows는 모든 EXIF 태그를 제거할 수 없습니다. 철저한 정리를 위해선 [브라우저 기반 도구](/ko/)를 이용하세요.

## macOS에서 EXIF 데이터 제거

1. **미리보기**에서 이미지 열기
2. **도구** → **인스펙터 표시**(⌘I)로 이동
3. **(i)** 탭 클릭, 그 후 **Exif** 탭 클릭
4. 하단의 **메타데이터 제거** 클릭

**제한 사항**: 미리보기는 모든 메타데이터 유형을 제거하지 않습니다. 완전 제거를 위해선 [온라인 도구](/ko/)를 이용하세요.

## 왜 브라우저 기반 제거가 최선인가요?

데스크톱 도구와 OS 기능에는 상당한 제한 사항이 있습니다:

| 방법 | GPS 제거 | 카메라 정보 제거 | 타임스탬프 제거 | 소프트웨어 설치 불필요 |
|------|:--------:|:----------------:|:---------------:|:---------------------:|
| 저희의 온라인 도구 | ✅ | ✅ | ✅ | ✅ |
| 아이폰 공유 | ✅ | ❌ | ❌ | ✅ |
| Windows 속성 | 부분 | 부분 | 부분 | ✅ |
| macOS 미리보기 | 부분 | 부분 | 부분 | ✅ |
| ExifTool (CLI) | ✅ | ✅ | ✅ | ❌ |

저희의 도구는 이미지 회전 방향이나 색상 프로필 같은 필수 데이터를 보존하면서 **모든** EXIF 메타데이터를 제거하므로, 사진이 올바르게 표시됩니다.

## 제거되는 것 vs 보존되는 것

**제거됨**: GPS 좌표, 카메라 제조사/모델, 타임스탬프, 시리얼 번호, 소프트웨어 정보, 섬네일, 저작권 정보

**보존됨**: 이미지 회전 방향(잘못된 회전 방지), 색상 프로필(정확한 색상 유지), 이미지 치수, 실제 픽셀 데이터

## 언제 EXIF 데이터를 제거해야 하나요?

항상 다음 전에 EXIF 데이터를 제거하세요:
- SNS에 사진 게시 (일부 플랫폼은 제거하지만 많지 않음)
- 이메일이나 메신저 앱으로 낯선 사람에게 사진 전송
- 포럼이나 중고 사이트에 사진 업로드
- 전문적인 맥락에서 사진 공유 (저널리즘, 법률)
- 콘테스트나 출판용 사진 제출

Les métadonnées photo sont des informations intégrées dans les fichiers image qui décrivent divers attributs de la photo. Comprendre ces métadonnées est essentiel pour protéger votre vie privée et gérer efficacement vos photos numériques.

## Types de métadonnées photo

### EXIF (Exchangeable Image File Format)
- Générées automatiquement par les appareils photo et smartphones
- Contiennent des données techniques de l'appareil (ouverture, vitesse d'obturation, ISO)
- Incluent les coordonnées GPS, les horodatages et les informations sur l'appareil
- Les plus pertinentes pour les préoccupations de confidentialité

### IPTC (International Press Telecommunications Council)
- Ajoutées manuellement ou par un logiciel d'édition photo
- Contiennent légende, mots-clés, crédits et informations de copyright
- Couramment utilisées par les organisations de presse et les agences de photothèque
- Peuvent contenir le nom et les coordonnées du photographe

### XMP (Extensible Metadata Platform)
- Développée par Adobe
- Format de métadonnées moderne, basé sur XML
- Peut contenir toutes les données EXIF et IPTC ainsi que des informations supplémentaires
- Utilisée par Adobe Lightroom, Photoshop et autres outils d'édition
- Peut inclure un historique d'édition étendu et des évaluations

## Quelles métadonnées se trouvent dans vos photos ?

Une photo de smartphone typique contient 20 à 50 champs de métadonnées :

| Catégorie | Champs | Risque de confidentialité |
|-----------|--------|---------------------------|
| Appareil | Marque, modèle, objectif | Moyen |
| Paramètres | Ouverture, vitesse d'obturation, ISO | Faible |
| GPS | Latitude, longitude, altitude | **Élevé** |
| Heure | Date de capture, date de modification | Moyen |
| Appareil | Numéro de série, logiciel | **Élevé** |
| Image | Dimensions, résolution, orientation | Faible |
| Miniature | Image aperçue intégrée | Moyen |

## Comment les métadonnées voyagent avec vos photos

Les métadonnées sont directement intégrées dans le fichier image. Cela signifie qu'elles voyagent avec la photo partout où elle va — lorsque vous envoyez par e-mail, message, téléversez, publiez, transférez ou partagez par n'importe quelle méthode.

## Comment visualiser les métadonnées photo

### En ligne (le plus simple)
Utilisez notre [afficheur EXIF gratuit](/fr/) — glissez-déposez n'importe quelle photo pour voir toutes ses métadonnées instantanément.

### Sur votre ordinateur
- **Windows** : Clic droit → Propriétés → Détails
- **macOS** : Clic droit → Obtenir les informations → Plus d'infos

## Gérer les métadonnées de vos photos

### Ce que vous devez supprimer
Avant de partager publiquement des photos, supprimez les coordonnées GPS, les numéros de série d'appareil, les horodatages, les informations personnelles et les miniatures intégrées.

### Ce que vous devez conserver
L'orientation de l'image (évite une rotation incorrecte), le profil de couleur (garde des couleurs précises) et les dimensions de l'image.

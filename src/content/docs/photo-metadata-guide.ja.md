写真メタデータとは、画像ファイル内に埋め込まれた情報であり、写真の様々な属性を表します。このメタデータを理解することは、プライバシーを保護し、デジタル写真を効果的に管理するために不可欠です。

## 写真メタデータの種類

### EXIF（Exchangeable Image File Format / 交換可能画像ファイル形式）
- カメラやスマートフォンによって自動生成される
- 技術的なカメラデータ（絞り値、シャッタースピード、ISO）を含む
- GPS座標、タイムスタンプ、デバイス情報を含む
- プライバシーに関する懸念で最も関連性が高い

### IPTC（International Press Telecommunications Council / 国際新聞通信評議会）
- 手動または写真編集ソフトウェアによって追加される
- キャプション、キーワード、クレジット、著作権情報を含む
- ニュース機関やストック写真エージェンシーで一般的に使用される
- 撮影者名や連絡先情報を含む可能性がある

### XMP（Extensible Metadata Platform / 拡張可能メタデータプラットフォーム）
- Adobeによって開発された
- 現代的なXMLベースのメタデータ形式
- すべてのEXIFおよびIPTCデータに加え、追加情報を含むことができる
- Adobe Lightroom、Photoshop、その他編集ツールで使用される
- 広範な編集履歴や評価を含むことができる

## 写真にはどんなメタデータが含まれている？

典型的なスマートフォンの写真には20〜50のメタデータフィールドが含まれています：

| カテゴリ | 項目 | プライバシーリスク |
|----------|--------|-------------|
| カメラ | メーカー、モデル、レンズ | 中 |
| 設定 | 絞り、シャッタースピード、ISO | 低 |
| GPS | 緯度、経度、高度 | **高** |
| 日時 | 撮影日、更新日 | 中 |
| デバイス | シリアル番号、ソフトウェア | **高** |
| 画像 | サイズ、解像度、向き | 低 |
| サムネイル | 埋め込みプレビュー画像 | 中 |

## メタデータは写真と一緒に移動する

メタデータは画像ファイルに直接埋め込まれます。つまり、メール送信、メッセージ送信、アップロード、投稿、転送、あらゆる方法で共有する際、写真と一緒に移動します。

## 写真メタデータの確認方法

### オンライン（一番簡単）
当社の[無料EXIFビューアー](/ja/)を使用——写真をドラッグ＆ドロップするだけで、すべてのメタデータを即座に確認できます。

### パソコン上で
- **Windows**: 右クリック → プロパティ → 詳細
- **macOS**: 右クリック → 情報を見る → 詳細

## 写真メタデータの管理

### 削除すべきもの
写真を公開で共有する前に、GPS座標、デバイスシリアル番号、タイムスタンプ、個人情報、埋め込みサムネイルを削除してください。

### 保持すべきもの
画像の向き（回転ミス防止）、カラープロファイル（色の正確性維持）、画像サイズ。

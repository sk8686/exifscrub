사진 메타데이터는 이미지 파일 내에 포함되어 사진의 다양한 속성을 설명하는 정보입니다. 이 메타데이터를 이해하는 것은 개인정보를 보호하고 디지털 사진을 효과적으로 관리하는 데 필수적입니다.

## 사진 메타데이터 유형

### EXIF (Exchangeable Image File Format)
- 카메라와 스마트폰에 의해 자동 생성
- 기술적 카메라 데이터 포함 (조리개, 셔터 스피드, ISO)
- GPS 좌표, 타임스탬프, 기기 정보 포함
- 개인정보 우려와 가장 관련이 높음

### IPTC (International Press Telecommunications Council)
- 수동 또는 사진 편집 소프트웨어에 의해 추가
- 캡션, 키워드, 크레딧, 저작권 정보 포함
- 뉴스 기관 및 스톡 사진 에이전시에서 널리 사용
- 사진작가 이름 및 연락처 정보 포함 가능

### XMP (Extensible Metadata Platform)
- Adobe에서 개발
- 현대적이고 XML 기반의 메타데이터 형식
- 모든 EXIF 및 IPTC 데이터와 추가 정보 포함 가능
- Adobe Lightroom, Photoshop 등 편집 도구에서 사용
- 광범위한 편집 기록 및 평점 포함 가능

## 사진에는 어떤 메타데이터가 있나요?

일반적인 스마트폰 사진에는 20-50개의 메타데이터 필드가 포함됩니다:

| 카테고리 | 필드 | 개인정보 위험 |
|----------|------|--------------|
| 카메라 | 제조사, 모델, 렌즈 | 중간 |
| 설정 | 조리개, 셔터 스피드, ISO | 낮음 |
| GPS | 위도, 경도, 고도 | **높음** |
| 시간 | 촬영 날짜, 수정 날짜 | 중간 |
| 기기 | 시리얼 번호, 소프트웨어 | **높음** |
| 이미지 | 치수, 해상도, 회전 방향 | 낮음 |
| 섬네일 | 포함된 미리보기 이미지 | 중간 |

## 메타데이터가 사진과 함께 이동하는 방식

메타데이터는 이미지 파일에 직접 포함됩니다. 이는 이메일, 메시지, 업로드, 게시, 전송, 또는 어떤 방식으로든 공유할 때마다 사진과 함께 이동한다는 뜻입니다.

## 사진 메타데이터 확인하는 방법

### 온라인 (가장 쉬움)
저희의 [무료 EXIF 뷰어](/ko/)를 사용하세요 — 어떤 사진이든 드래그 앤 드롭하면 모든 메타데이터를 즉시 확인할 수 있습니다.

### 컴퓨터에서
- **Windows**: 우클릭 → 속성 → 세부사항
- **macOS**: 우클릭 → 정보 가져오기 → 추가 정보

## 사진 메타데이터 관리

### 제거해야 할 것
사진을 공개적으로 공유하기 전에 GPS 좌표, 기기 시리얼 번호, 타임스탬프, 개인정보, 포함된 섬네일을 제거하세요.

### 보존해야 할 것
이미지 회전 방향(잘못된 회전 방지), 색상 프로필(정확한 색상 유지), 이미지 치수.

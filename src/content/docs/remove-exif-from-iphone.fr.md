Les photos iPhone contiennent des métadonnées EXIF extensives, incluant les coordonnées GPS, les détails de l'appareil photo et les horodatages. Apple a ajouté certains outils intégrés pour gérer ces données, mais ils présentent des limites importantes.

## Quelles données EXIF contiennent les photos iPhone ?

Une photo iPhone comprend généralement les coordonnées GPS, le modèle d'appareil photo, les informations sur l'objectif, les paramètres d'exposition, les informations de profondeur, les informations HDR, les horodatages, la version du logiciel et l'orientation de l'image.

## Méthode 1 : Suppression de localisation intégrée iOS

Depuis iOS 13, vous pouvez supprimer les données de localisation lors du partage :

1. Ouvrez l'application **Photos**
2. Sélectionnez la ou les photos que vous souhaitez partager
3. Appuyez sur le bouton **Partager**
4. Appuyez sur **Options** en haut de la feuille de partage
5. Passez **Localisation** sur **Désactivé**
6. Partagez la photo

**Limites :** Supprime uniquement les données GPS — le modèle d'appareil, les horodatages et les autres métadonnées restent.

## Méthode 2 : Supprimer la localisation d'une seule photo

Sous iOS 17+, vous pouvez supprimer la localisation d'une photo sauvegardée :

1. Ouvrez la photo dans **Photos**
2. Balayez vers le haut ou appuyez sur le bouton **(i)**
3. Appuyez sur **Ajuster** à côté de la localisation
4. Appuyez sur **Aucune localisation**

**Limites :** Supprime uniquement le GPS — toutes les autres données EXIF restent.

## Méthode 3 : Désactiver complètement le géotagging

1. Allez dans **Réglages** → **Confidentialité et sécurité** → **Services de localisation**
2. Trouvez **Appareil photo** dans la liste
3. Sélectionnez **Jamais**

**Limites :** N'affecte pas les photos existantes.

## Méthode 4 : Utilisez notre outil en ligne gratuit (Recommandé)

Pour une suppression complète des EXIF des photos iPhone :

1. Ouvrez [notre EXIF Scrub](/fr/) dans Safari ou Chrome sur votre iPhone
2. Sélectionnez ou glissez vos photos
3. Voyez toutes les métadonnées que contiennent vos photos
4. Appuyez sur « Nettoyer » pour tout supprimer
5. Téléchargez les photos nettoyées

**Avantages :** Supprime TOUTES les métadonnées, pas seulement le GPS. Pas d'installation d'application requise. Fichiers traités localement. Fonctionne avec les photos HEIC.

## HEIC vs JPEG sur iPhone

Les photos iPhone sont sauvegardées au format HEIC par défaut (depuis iOS 11). Notre outil gère automatiquement les fichiers HEIC — il les convertit en JPEG tout en supprimant toutes les métadonnées.

Pour configurer votre iPhone pour prendre des photos en JPEG : **Réglages** → **Appareil photo** → **Formats** → **Plus compatible**

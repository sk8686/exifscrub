Si vous avez déjà pris une photo avec votre smartphone, il y a de fortes chances qu'elle contienne vos coordonnées GPS exactes — précises à quelques mètres près. Cela signifie que toute personne avec qui vous partagez cette photo peut voir exactement où elle a été prise.

## Comment les données GPS se retrouvent-elles dans les photos ?

Lorsque vous prenez une photo, l'application photo de votre téléphone lit votre localisation GPS actuelle depuis les services de localisation de votre appareil et l'intègre directement dans le fichier image sous forme de métadonnées EXIF. Cela se fait automatiquement sur la plupart des smartphones, sauf si vous l'avez spécifiquement désactivé.

Les données GPS comprennent généralement :
- **Latitude et longitude** — votre position exacte sur Terre
- **Altitude** — votre hauteur au-dessus du niveau de la mer
- **Horodatage GPS** — moment où la localisation a été enregistrée
- **Direction** — vers quoi pointait l'appareil photo (sur certains appareils)

## Que peuvent révéler les données GPS ?

Une seule photo géolocalisée peut révéler :
- Votre **adresse domicile** (photos prises chez vous)
- Votre **lieu de travail** (photos prises au bureau)
- **L'école de vos enfants** (photos lors d'événements scolaires)
- Votre **routine quotidienne** (photos prises aux mêmes endroits de manière répétée)
- Vos **modèles de voyage** (photos de vacances avec données de localisation)
Des **localisations sensibles** que vous avez visitées (établissements médicaux, cabinets d'avocats, etc.)

## Comment vérifier si vos photos contiennent des données GPS

**Vérification rapide** : utilisez notre [afficheur EXIF gratuit](/fr/) — glissez-déposez votre photo pour voir instantanément si elle contient des coordonnées GPS.

**Sur iPhone** : Ouvrez Photos → sélectionnez une photo → balayez vers le haut → cherchez une carte ou un nom de localisation

**Sur Android** : Ouvrez Galerie → sélectionnez une photo → appuyez sur Infos → cherchez les données de localisation

## Comment supprimer le GPS des photos

### Méthode 1 : Utilisez notre outil gratuit (Recommandé)

1. Rendez-vous sur notre [EXIF Scrub](/fr/)
2. Glissez-déposez vos photos
3. Voyez les coordonnées GPS mises en évidence comme un risque de confidentialité
4. Cliquez sur « Nettoyer » pour supprimer toutes les données de localisation
5. Téléchargez les photos nettoyées

Vos fichiers ne quittent jamais votre appareil — tout le traitement se fait localement dans votre navigateur.

### Méthode 2 : Désactiver le géotagging sur iPhone

1. Allez dans **Réglages** → **Confidentialité et sécurité** → **Services de localisation**
2. Trouvez **Appareil photo** dans la liste des applications
3. Changez pour **Jamais** ou **Pendant l'utilisation de l'application** (et refusez lorsque demandé)

### Méthode 3 : Désactiver le géotagging sur Android

1. Ouvrez l'application **Appareil photo**
2. Appuyez sur **Réglages** (icône d'engrenage)
3. Désactivez **Localisation** ou **Géotagging**

## Le problème du « simplement désactiver le GPS »

Même si vous désactivez le géotagging maintenant, votre bibliothèque de photos existante contient encore des données GPS dans chaque photo que vous avez prise avec les services de localisation activés. Et certaines applications rajoutent des données de localisation lorsque vous partagez des photos.

La seule méthode fiable pour garantir que les données GPS sont supprimées est de [les retirer du fichier photo lui-même](/fr/) avant de partager.

## Ne faites pas confiance aux réseaux sociaux pour supprimer la localisation

Certaines plateformes prétendent supprimer les données EXIF lorsque vous téléversez des photos, mais :
- **Facebook** supprime les EXIF des images affichées mais peut les conserver en interne
- **Instagram** supprime les EXIF des publications mais pas des messages directs
- **Twitter/X** supprime la plupart des données EXIF
- **WhatsApp** conserve les données EXIF des photos partagées
- **L'e-mail** conserve toutes les données EXIF
- **La plupart des forums** conservent toutes les données EXIF

L'approche la plus sûre : supprimez vous-même les données GPS avant de partager, quelle que soit la plateforme.

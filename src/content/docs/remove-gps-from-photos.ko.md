스마트폰으로 사진을 찍어본 적이 있다면, 정확한 GPS 좌표가 포함되어 있을 가능성이 높습니다 — 몇 미터 이내의 정밀도로요.这意味着 사진을 공유하는 누구나 정확히 어디에서 찍혔는지 알 수 있다는 뜻입니다.

## GPS 데이터는 어떻게 사진에 들어가나요?

사진을 찍을 때, 스마트폰의 카메라 앱은 기기의 위치 서비스에서 현재 GPS 위치를 읽어 EXIF 메타데이터로 이미지 파일에 직접 기록합니다. 특별히 비활성화하지 않았다면 대부분의 스마트폰에서 자동으로 발생합니다.

GPS 데이터는 일반적으로 다음을 포함합니다:
- **위도와 경도** — 지구상의 정확한 위치
- **고도** — 해발 얼마나 높았는지
- **GPS 타임스탬프** — 위치가 기록된 시간
- **방향** — 카메라가 향하고 있는 방향 (일부 기기에서)

## GPS 데이터는 무엇을 드러낼 수 있나요?

지오태깅된 단 한 장의 사진도 다음을 드러낼 수 있습니다:
- **집 주소** (집에서 찍힌 사진)
- **직장** (사무실에서 찍힌 사진)
- **자녀의 학교** (학교 행사에서 찍힌 사진)
- **일상 루틴** (같은 장소에서 반복적으로 찍힌 사진)
- **여행 패턴** (위치 데이터가 있는 휴가 사진)
- 방문한 **민감한 장소** (의료 시설, 법률 사무소 등)

## 사진에 GPS 데이터가 있는지 확인하는 방법

**빠른 확인**: 저희의 [무료 EXIF 뷰어](/ko/)를 사용하세요 — 사진을 드래그 앤 드롭하면 GPS 좌표 포함 여부를 즉시 확인할 수 있습니다.

**아이폰**: 사진 열기 → 사진 선택 → 위로 스와이프 → 지도나 위치 이름 확인

**안드로이드**: 갤러리 열기 → 사진 선택 → 정보 탭 → 위치 데이터 확인

## 사진에서 GPS 제거하는 방법

### 방법 1: 무료 도구 사용 (권장)

1. 저희의 [EXIF Scrub](/ko/)로 이동
2. 사진을 드래그 앤 드롭
3. 개인정보 위험으로 강조된 GPS 좌표 확인
4. "Clean" 클릭하여 모든 위치 데이터 제거
5. 정리된 사진 다운로드

파일은 절대 기기를 벗어나지 않습니다 — 모든 처리가 브라우저에서 로컬로 이루어집니다.

### 방법 2: 아이폰에서 지오태깅 비활성화

1. **설정** → **개인정보 및 보안** → **위치 서비스**로 이동
2. 앱 목록에서 **카메라** 찾기
3. **사용 안 함** 또는 **앱을 사용하는 동안**으로 변경 (프롬프트 시 거부)

### 방법 3: 안드로이드에서 지오태깅 비활성화

1. **카메라** 앱 열기
2. **설정**(톱니바퀴 아이콘) 탭
3. **위치** 또는 **지오태깅** 끄기

## "그냥 GPS 끄기"의 문제점

지금 지오태깅을 비활성화하더라도, 기존 사진 라이브러리에는 위치 서비스가 활성화된 상태에서 찍은 모든 사진에 여전히 GPS 데이터가 포함되어 있습니다. 또한 일부 앱은 사진을 공유할 때 위치 데이터를 다시 추가합니다.

GPS 데이터가 제거되었는지 확실히 하는 유일한 신뢰할 수 있는 방법은 공유 전에 [사진 파일 자체에서 제거](/ko/)하는 것입니다.

## SNS가 위치 데이터를 제거한다고 믿지 마세요

일부 플랫폼은 사진 업로드 시 EXIF 데이터를 제거한다고 하지만:
- **Facebook**은 표시되는 이미지에서 EXIF를 제거하지만 내부적으로 보존할 수 있음
- **Instagram**은 게시물에서 EXIF를 제거하지만 직접 메시지에서는 제거하지 않음
- **Twitter/X**는 대부분의 EXIF 데이터를 제거
- **WhatsApp**은 공유 사진에서 EXIF 데이터 보존
- **이메일**은 모든 EXIF 데이터 보존
- **대부분의 포럼**은 모든 EXIF 데이터 보존

가장 안전한 접근법: 플랫폼과 관계없이 공유 전에 직접 GPS 데이터를 제거하세요.
